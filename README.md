# Data Analytics Hub

## Setup

1. Clone repository
2. Run `bash bin/deploy.sh`
3. Access app: http://127.0.0.1:5000
4. Access Minio: http://127.0.0.1:9000 (username: minioadmin, password: minioadmin)

## Endpoints
- `/health` : App health
- `/storage/health` : Minio storage health
- `/data` : Upload/GET/Delete JSON files
